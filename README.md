
# SimCLR-Lite on CIFAR-10 (Self-Supervised Deep Learning)

This project implements a **self-supervised contrastive learning pipeline (SimCLR)**
using PyTorch on the CIFAR-10 dataset.

## Features
- SimCLR self-supervised pretraining
- Linear probe evaluation
- t-SNE visualization
- Clean, academic codebase

Suitable for **research portfolios and PhD applications**.
